# AI CUP ESG Final Project

## Project Description
This project is for the AI CUP 2026 ESG commitment verification task.  
The goal is to predict four labels:

1. promise_status
2. evidence_status
3. evidence_quality
4. verification_timeline

## Environment
- Google Colab
- Python
- pandas
- numpy
- scikit-learn
- PyTorch
- Hugging Face Transformers

## Files
- `2026AICUP期末.ipynb`: main training and prediction code

## How to Run
1. Open the notebook in Google Colab.
2. Upload the train, validation, and test CSV files.
3. Run all cells.
4. The program will generate `submission.csv`.
5. Upload `submission.csv` to the AIDEA platform.
